# Car Rental Management System

The Car Rental Management System is a console-based java application that allows users to rent and return cars.
It is designed to demonstrate core java and object oriented programming concepts using a real-world use case.
The system manages cars, customers, and rental details efficiently through a menu-driven interface.

## Features
-Add and manage cars
-Rent available cars
-Return rented cars
-Calculate total rental price based on number of days
-Menu-driven user interaction
-Maintains rental records using collections


##Technologies used

-Java(Core java)
-Object oriented programming
-Java collections
-Scanner for user input


##Project Structure

Car.java //Represents a car
Customer.java //Represents a customer
Rental.java //Represents a rental record
CarRentalSystem.java //Core system logic
Main.java //Application entry point

##Application flow
1.User selects an option from the menu.
2.System displays available cars.
3.User rents or returns a car.
4.Rental details are updated accordingly.
